corrosion panel

corrosion panel is a small wayland sidebar written in python using gtk-layer-shell.
it hides at the edge of the screen and slides out when the mouse touches it.

features

auto-hide until mouse hover

plugin system for adding widgets

css-based theming

minimal dependencies

install

sudo apt install python3-gi gir1.2-gtk-4.0 gir1.2-gtklayer
git clone https://github.com/travtherobber/corrosion-panel.git
